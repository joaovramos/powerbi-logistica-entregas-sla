# Logística — Entregas e SLA (Power BI)

## Objetivo
Analisar dados de logística e supply chain, acompanhando performance
de entregas, cumprimento de SLA, custos por rota e eficiência operacional.

## Contexto do Projeto
Projeto demonstrativo utilizando dados públicos e estruturados
para simular cenários reais de logística.

## Escopo Analítico
- Entregas realizadas
- Prazo vs realizado
- Custos logísticos
- Análise por rota e transportadora

## Principais Indicadores
- SLA (%)
- Lead time médio
- Entregas no prazo
- Custo por entrega
- Atrasos por rota

## Ferramentas Utilizadas
- Power BI Desktop
- Power Query (M)
- Excel / CSV

## Observações
Fonte de dados pública (Kaggle) utilizada apenas para fins educacionais.
